CBMG 2015

This is material for a two week intensive course on programming and scientific computing I developed and lead in summer 2015 for math grad students at University of Illinois at Urbana-Champaign.

If you find it useful, please let me know! I'd love to hear that others have benefited from it beyond just my class. Sadly, the original notebooks have been lost, so only the HTML renderings remain.

Noice

Streamlit example of generating cool images from random noise. To run it, you'll need make sure you have the required packages, then you can use "streamlit run" directly from the web.

pip3 install numpy scipy matplotlib streamlit
streamlit run https://raw.githubusercontent.com/seanshahkarami/seanshahkarami.github.io/master/noice.py
