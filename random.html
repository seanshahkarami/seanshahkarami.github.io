Noice

Streamlit example of generating cool images from random noise. To run it, you'll need make sure you have the required packages, then you can use "streamlit run" directly from the web.

pip3 install numpy scipy matplotlib streamlit
streamlit run https://raw.githubusercontent.com/seanshahkarami/seanshahkarami.github.io/master/noice.py
